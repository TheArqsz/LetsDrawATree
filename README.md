# Lets Draw A Tree

A few (hopefully) helpful scripts for the 1st year of Cybersecurity at WUST

## Prerequisites
Scripts are written under **c++**

Before you can use it you need to have:
* gcc/g++

## Usage

To compile:

```
g++ <script>.cpp
```

To run:

```
./a.exe
```

or 

```
chmod +x ./a.out
./a.out
```
